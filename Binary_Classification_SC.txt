Environment & Requirements
============================================================

This notebook reproduces the 10-fold cross-validation, hold-out
evaluation, and external-dataset validation of the Symbolic
Classifier (SC) for pediatric pneumonia detection on chest X-rays.

The radiomic feature CSVs (e.g., train2_*.csv, test2_*.csv,
normal_external.csv, patient_external.csv) are produced upstream
in MATLAB and are assumed to already exist in the working directory.

Tested with Python 3.10.

Essential Python libraries
--------------------------
numpy
pandas
scipy
scikit-learn
matplotlib
pysr

Notes on PySR
-------------
PySR is a Python wrapper around a Julia backend. After
`pip install pysr`, on first run PySR will install/compile the
required Julia packages automatically. A working Julia
installation (>= 1.9) is required.

Quick install
-------------
pip install numpy pandas scipy scikit-learn matplotlib pysr

Recommended versions
--------------------
numpy        >= 1.24
pandas       >= 2.0
scipy        >= 1.10
scikit-learn >= 1.3
matplotlib   >= 3.7
pysr         >= 0.18

Run order
---------
1. Place all input CSVs (train2_*, test2_*, normal_external.csv,
   patient_external.csv) in the same folder as the notebook,
   or update DATA_DIR in the configuration cell.
2. Run the cells top-to-bottom.
3. Outputs (cv10_pysr_raw.csv, cv10_pysr_equations.csv) are
   written to OUT_DIR.

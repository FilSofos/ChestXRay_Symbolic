# ChestXRay_Symbolic
Code for ChestXRay image dataset classification with Symbolic Regression methods. The full dataset is taken from the original work of 
"Kermany DS, Goldbaum M, Cai W, Valentim CCS, Liang H, Baxter SL, et al.,Identifying Medical Diagnoses and Treatable Diseases by Image-Based Deep Learning. Cell. 2018;172(5):1122-31.e9. https://doi.org/10.1016/j.cell.2018.02.010."

The Matlab file can be adjusted to read any kind of dataset. The Jupyter file can be also adjusted to search for equations with different performance measures, different operands etc. See the gplearn site for more info.
